About This Predictor

This web application is a demo of a customer churn prediction model. The model is based on an artificial neural network trained on a telecommunications customer dataset.

Project Objective

The primary goal of this project is to develop a predictive model that can identify customers at risk of churning (i.e., canceling their subscription). By predicting churn, a company can proactively intervene with retention strategies to prevent customer loss.

Dataset

The model was trained on a dataset containing information about a telecommunications company's customers. The dataset includes a variety of demographic and service-related features, such as gender, tenure, internet service type, contract type, and monthly charges.

Data Preprocessing

To prepare the data for the neural network, several key preprocessing steps were performed:

  • **Missing Values:** Rows with missing values in the `TotalCharges` column were removed from the dataset.
  • **Feature Transformation:** The model requires numerical inputs. Binary categorical features like `Partner`, `Dependents`, `PhoneService`, etc., were converted to numerical values (`Yes` -> `1`, `No` -> `0`). Similarly, `gender` was mapped to `Female` -> `1` and `Male` -> `0`.
  • **One-Hot Encoding:** For features with more than two categories (e.g., `InternetService`, `Contract`, `PaymentMethod`), one-hot encoding was used to create new binary columns for each category.
  • **Feature Scaling:** Numerical features (`tenure`, `MonthlyCharges`, `TotalCharges`) were scaled using a `MinMaxScaler` to bring all values into a range between 0 and 1. This prevents features with larger scales from dominating the model's training process.

Model Architecture

The prediction model is a simple sequential neural network built using the TensorFlow/Keras library. It consists of the following layers:

  • **Input Layer:** A dense layer with 26 neurons, corresponding to the number of features after preprocessing. It uses a `relu` activation function.
  • **Hidden Layer:** A dense layer with 15 neurons and a `relu` activation function, allowing the model to learn complex patterns in the data.
  • **Output Layer:** A single-neuron dense layer with a `sigmoid` activation function. This layer outputs a value between 0 and 1, which can be interpreted as the probability of a customer churning. A probability greater than 0.5 is classified as churn.

The model was compiled with the `adam` optimizer and `binary_crossentropy` loss function, and it was trained for 100 epochs.
